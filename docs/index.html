bSSG Docs

bSSG is a simple static site generator written in Python. It accepts Markdown content and HTML templates and combines them together into a full static site, alongside a few helpful utilities to make it easier. The project is licensed under the MIT License and is open-source, so feel free to contribute in our GitHub repository!

You can read the installation guide to get it installed and then read the quickstart guide to learn the basics of bSSG. A full reference manual will be available soon.
